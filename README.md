# CESPR

Projeto de controle de peças recondicionadas desenvolvido como MPV para a disciplina de Engenharia de Software / Banco de Dados.

## Visão Geral  
O sistema CESPR tem como objetivo dar suporte à organização e rastreabilidade de peças devolvidas e recondicionadas, bem como pendências com clientes e vendedores. Permite modelar clientes, modelos de peças, pedidos, vendedores e suas relações, além de manter o histórico no banco de dados.

## Tecnologias  
- Java (classes: `Cliente.java`, `Modelo.java`, `Pedido.java`, `Pedido_Modelo.java`, `Pessoa.java`, `Vendedor.java`…)  
- Banco de dados SQL (arquivo `CESPR_db.sql`)  
- Modelo relacional (tabelas / entidades conforme regras de negócio)
